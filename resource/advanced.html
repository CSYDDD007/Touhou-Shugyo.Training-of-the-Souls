Usage of nodes

Thread: Create a thread with independent time counter.
Allow the program execute two or more things at the same time.

Repeat: Create a repeat nodes
Support variable of repeat and increment

Code: Create a code node
Execute the code while the program is running

Tweening: Create a tweening node
Change the specific variable with tweening
self: if the node is insert inside a defined class, it refers to the class itself
last: the last created object
