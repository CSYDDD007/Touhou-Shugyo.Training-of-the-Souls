PySTG Introduction

A very simple Python project that references to LuaSTG:)

Actually, many functions in LuaSTG are not appeared in this project(My coding skill is still poor)

However, at least it can support basic functions as being an Editor of STG game

The Menu Bar

Similar to LuaSTG, PySTG uses nodes to control the game logic
Such as enemy, bullet, boss, background, BGM
There are there options in [File]:

  • New: create an new project template
  • Load: load an existed file(sba_dat)
  • Save: save current project as sba_dat

There are three options in [Help]:
  • Introduction
  • Basic
  • Advanced
There is an option in [Test]:
  • Test: execute the game based on Editor
While running the game, PySTG is locked for safety:)
Remember to place the (sba_dat) files in the 'mods' folder
The game reads the files from the 'mods' folder
(Actually, sba_dat files are saved as json format)
