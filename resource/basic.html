PySTG Basic

Structure of the project

There are 3 folders at the beginning:

  • Resource(Use for load resources)
  • Classes(Use for deinfe object, background, enemy, bullet, boss)
  • Main(Use for control the game process)

ToolBar

There are totally 10 genres in the ToolBar:

  • General
  • Stage
  • Times
  • Enemy
  • Boss
  • Bullet
  • Audio
  • Background
  • Resource
  • Object

General

There are 6 nodes in General Type:

  • Folder: Use for putting other nodes
  • Code: Write Python Code to control the game more flexible
  • Comment: Use for Documentory
  • If: branch
  • Else If: still branch, need to be placed on the bottom of [If node]
  • Else: also branch, need to be placed on the bottom of [If node] or [Else If node]

Stage

There are 3 nodes in Stage Type:

  • Rank: Define a rank, need to be placed inside the [main folder]
  • Stage: Define a stage, need to be placed inside the [Rank node]
  • Stage Title: Show the title of stage

Times

There are 5 nodes in Times Type:

  • Thread: Create a thread, use for independent time
  • Wait: Create a pause between nodes
  • Repeat: Create a repeat
  • Break: Break the current repeat, need to have a [Repeat node] father
  • Continue: Skip the current repeat step, need to have a [Repeat node] father

Enemy

There are 2 nodes in Enemy Type:

  • Create Enemy: Create a defined enemy
  • Enemy Class: Define a enemy class, need to be placed inside the [Classes folder]

Boss

There are 6 nodes in Boss Type:

  • Create Boss: Create a define boss
  • Boss Class: Define a boss class, need to be placed inside the [Classes folder]
  • SpellCard: Define a SpellCard, need to be placed inside the [Boss Class node]
  • Dialog Group: Define a Dialog Group, need to be placed inside the [Boss Class node]
  • Dialog: Create a dialog, need to be placed inside the [Dialog Group node]
  • Move to: Call the boss move to somewhere, need to have a [Boss Class node] father

Bullet

There are 3 nodes in Bullet Type:

  • Create Simple Bullet: Create a simple bullet
  • Create Bullet: Create a defined bullet
  • Bullet Class: Define a bullet class, need to be placed inside the [Classes folder]

Audio

There are 2 nodes in Audio Type:

  • Sound: Play sound
  • Music: Play BGM

Background

There are 4 nodes in Background Type:

  • Create Background: Create a defined background
  • Clear Backgrounds: Clear all backgrounds
  • Background Class: Define a background class, need to be placed inside the [Classes folder]
  • Layer Class: Define a layer class, need to be placed inside the [Background Class node]

Resource

There are 3 nodes in Resource Type:

  • Load Image: load image from files, need to be placed inside the [Resource folder]
  • Load Sound: load sound from files, need to be placed inside the [Resource folder]
  • Load Music: load music from files, need to be placed inside the [Resource folder]

Object

There are 5 nodes in Object Type:

  • Variable: Set a variable
  • Velocity: Change the velocity of something
  • Tweening: Call a tweening to a variable
  • Create Object: Create a defined object
  • Object Class: Define a object class

Overall Bar

There are 3 buttons in Overall Bar

  • Copy: Copy a selected node with its children
  • Paste: Paste the copied nodes into the project(according the insert mode)
  • Delete: Delete a selected node with its children

How to insert nodes?

There are 3 selection buttons in the Overall Bar

  • Insert Up Mode: insert a node/nodes on the top of a selected node
  • Insert Inside Mode: insert a node/nodes inside a selected node(as the children of selected node)
  • Insert Down Mode: insert a node/nodes on the bottom of a selected node
